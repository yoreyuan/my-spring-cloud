# 1 Consul简介
Consul是一个分布式高可用的服务网络（service mesh）解决方案，提供包括服务发现、配置和分段功能在内的全功能控制平面。
Consul是一个分布式高可用的系统服务发现与配置工具。简单来说，它跟Eureka的核心功能一样，但也略有不同([https://www.consul.io/intro/vs/index.html](https://www.consul.io/intro/vs/index.html))：
* Consul使用go语言编写，与HTTP方式对外提供服务。
* Consul支持多数据中心，这是它的一大特点
* Consul处理服务发现之外，还有一些别的功能
* Consul的一致性协议是Raft

Consul官网：https://www.consul.io/
Consul GitHub: https://github.com/HashiCorp/consul
